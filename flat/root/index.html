How to use

Use two finger drag or mouse wheel to zoom

Drag by mouse or finger

Tap/click percussion pane to set/remove beat

Tap/click instrument name to select instrument

Tap/click instrument pane to start/remove note

Tap/click 'Repeat ...' to clone selected track

Tap/click arrows to transpose measure

Tap/click double-arrows to transpose track

Tap/click 'Swap' to swap tracks

Tap/click equalizer bars to change tone

Tap/click instrument or drum button to select sample

Select 'File' for import/export MIDI etc

Select 'Share riff' to share via Twitter, Facebook, VK etc

Select 'Clear all' to start new song
